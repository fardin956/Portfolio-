What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls how HTML elements look — including colors, sizes, fonts, margins, and even animations.

How to Add CSS

  • Inline: Inside an HTML tag using style=""
  • Internal: Inside a <style> tag in the <head>
  • External: In a separate .css file
<style>
  h1 {
    color: blue;
  }
</style>

CSS Syntax

CSS uses a selector and a declaration block: 

selector {
  property: value;
}

Example: 

p {
  color: red;
  font-size: 16px;
}

What is a CSS Property?

A property is the styling feature you want to change. For example: color, font-size, margin, etc. 

h1 {
  color: green;      /* Property: color */
  font-weight: bold; /* Property: font-weight */
}

What is a CSS Attribute?

In CSS, the word "attribute" usually refers to HTML attributes that CSS can select using selectors. For example: 

a[target="_blank"] {
  color: red;
}

This targets anchor (<a>) elements with a target="_blank" attribute.

Selectors

  • * → Universal
  • tag → Element selector
  • .class → Class selector
  • #id → ID selector
  • [attr] → Attribute selector

Colors in CSS

body {
  background-color: #f0f0f0;
  color: #333;
}

Text Styling

h1 {
  font-family: Arial;
  font-size: 24px;
  text-align: center;
}

Box Model

Every element is a box made up of:

  • Content
  • Padding
  • Border
  • Margin
div {
  padding: 10px;
  border: 2px solid black;
  margin: 20px;
}

Positioning

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky
.box {
  position: absolute;
  top: 50px;
  left: 100px;
}

Flexbox Basics

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Grid Basics

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}

Responsive Design

Use media queries to make layouts responsive:

@media (max-width: 600px) {
  body {
    background: lightblue;
  }
}

Conclusion

CSS controls the look and feel of your website. Once you're confident with properties, selectors, and layout systems like Flexbox and Grid, you can build beautiful web pages!
