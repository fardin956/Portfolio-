Complete Guide to HTML

Author: Fardin Hussain | Date: April 2025

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures the content and elements of a website, acting as the "skeleton" before CSS and JavaScript are added for design and functionality.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

Explanation:

  • <!DOCTYPE html>: Declares the document type (HTML5).
  • <html>: Root element of the page.
  • <head>: Contains meta info, title, links, scripts.
  • <title>: Sets the browser tab title.
  • <body>: Contains visible page content.

HTML Text Formatting Tags

  • <b> or <strong>: Bold text
  • <i> or <em>: Italic text
  • <u>: Underlined text
  • <mark>: Highlighted text
  • <sub>: Subscript
  • <sup>: Superscript
  • <small>: Smaller text

Example:

<p>This is <strong>bold</strong> and <em>italic</em> text.</p>

HTML Headings and Paragraphs

  • <h1> to <h6>: Headings from largest to smallest.
  • <p>: Paragraph tag for blocks of text.

Lists in HTML

  • <ul>: Unordered list
  • <ol>: Ordered list
  • <li>: List item

Example:

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

HTML Links

<a>: Defines a hyperlink.

<a href="https://example.com" target="_blank">Visit Example</a>

HTML Images

<img>: Embeds images.

<img src="image.jpg" alt="Description" width="300" />

Tables in HTML

  • <table>: Table element
  • <tr>: Table row
  • <th>: Table header
  • <td>: Table data
<table border="1">
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>22</td></tr>
</table>

Forms in HTML

  • <form>: Defines a form
  • <input>: Input field
  • <label>: Label for input
  • <textarea>, <button>, <select> also used
<form>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <input type="submit" value="Submit">
</form>

Semantic Tags

  • <header>, <footer>
  • <nav>, <main>
  • <section>, <article>
  • <aside>

These tags help describe the meaning and structure of the content.

Multimedia

Audio

<audio controls>
  <source src="sound.mp3" type="audio/mpeg">
</audio>

Video

<video width="320" height="240" controls>
  <source src="video.mp4" type="video/mp4">
</video>

Common HTML Attributes

  • id: Unique identifier for an element
  • class: Used to apply CSS
  • src: Specifies image, audio, or video source
  • href: Hyperlink reference
  • style: Inline CSS styles
  • title: Tooltip text
  • alt: Alternate text for media

Special Characters in HTML

  • &lt; → <
  • &gt; → >
  • &amp; → &
  • &quot; → "
  • &nbsp; → non-breaking space

Conclusion

HTML is the essential foundation of all web pages. Once you're confident with HTML, you can level up with CSS for styling and JavaScript for interactivity.
