What is JavaScript?

JavaScript is a high-level, dynamic programming language used to create interactive and dynamic features on websites. It works alongside HTML and CSS, enabling features like sliders, forms, games, and more.

How JavaScript Works

JavaScript code runs in the browser. It can respond to user actions like clicking, typing, or scrolling. It can also manipulate HTML elements and change CSS styles dynamically.

JavaScript Variables

Variables store data that your program can use or change. Use let, const, or var to declare them. 

let name = "Alice";
const age = 18;
var isStudent = true;

Data Types

JavaScript supports several data types:

  • String: Text values — "Hello"
  • Number: Numeric values — 100
  • Boolean: True or false — true
  • Array: List of values — [1, 2, 3]
  • Object: Key-value pairs — { name: "Ali", age: 20 }
  • Null / Undefined: Empty or unknown values

What are Properties in JavaScript?

A property is a value that belongs to an object. It's like a variable attached to the object.

Example:

let car = {
  brand: "Toyota",
  color: "red",
  speed: 120
};

console.log(car.color); // "red"

Here, brand, color, and speed are properties of the object car.

What are Attributes in JavaScript?

Technically, attributes belong to HTML elements and are accessed using JavaScript through the DOM.

Example:

<img id="photo" src="image.jpg" alt="My Photo">

Here, src and alt are attributes of the <img> element. JavaScript can read and change them using getAttribute() and setAttribute().

JavaScript Functions

Functions are reusable blocks of code that perform a task. You can call them whenever needed. 

function greet(name) {
  console.log("Hello, " + name);
}
greet("Ravi");

Conditionals

Use if, else if, and else to make decisions.

let score = 80;
if (score >= 90) {
  console.log("Grade A");
} else if (score >= 75) {
  console.log("Grade B");
} else {
  console.log("Try again");
}

Loops

Loops help you run code multiple times.

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

DOM Manipulation

DOM (Document Object Model) allows you to access and change HTML using JavaScript. 

document.getElementById("title").style.color = "blue";

Events

React to user actions like clicks, key presses, or hovers.

document.getElementById("btn").addEventListener("click", function() {
  alert("Button Clicked!");
});

Conclusion

JavaScript brings life to your web pages. Start small, build projects, and practice regularly to master the language.
